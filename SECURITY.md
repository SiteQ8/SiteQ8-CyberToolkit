# Security Policy

## Supported Versions

This is an educational platform. We recommend always using the latest version from the main branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this educational platform, please follow these steps:

### For Critical Security Issues

1. **DO NOT** open a public issue
2. Email security details to: Site@hotmail.com
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

### For Non-Critical Issues

1. Open an issue on GitHub
2. Tag it with `security` label
3. Provide detailed description
